海外の店では入店した瞬間に店員から声をかけられるのが普通で、ここで固まる人が多い。
この記事では店員への自然な返し方から、探し物・試着・会計・返品までの定番表現をまとめる。

## 店員に話しかけられたら

海外の接客は「声をかけるのが仕事」なので、話しかけられても買う義務は一切ない。まずはこの返しを口癖にする。

- **I'm just looking, thanks.** — 見ているだけです、ありがとう（これが大定番）
- **I'm just browsing.** — ちょっと見て回っているだけです
- **Thanks, I'll let you know if I need anything.** — 何かあったら声をかけますね
- **Actually, yes — I'm looking for a jacket.** — ちょうど探していて — ジャケットを見ています

> 💡 店員の **Can I help you find anything?**（お探し物はありますか?）には、不要なら **I'm just looking, thanks.**、あれば **Yes, actually — do you have...?** と返すのが自然。無言で首を振るのは感じが悪いので、**thanks** を一言添えるのがコツ。

> 🧭 日本の感覚で店員を無視したり会釈だけで済ませたりしがちだが、英語圏では **Hi!** と返してから **Just looking, thanks.** と言うのが普通。あいさつを返さないほうが不自然に映る。

## 探し物を伝える

- **Do you have this in a medium?** — これのMサイズはありますか
- **Do you carry phone cases?** — スマホケースは置いていますか（その店で「取り扱いがあるか」）
- **Where can I find the socks?** — 靴下はどこにありますか
- **I'm looking for a gift for my friend.** — 友人へのプレゼントを探しています
- **Do you have this in stock?** — これの在庫はありますか

サイズ・色違いを聞くときの型はこれだけ覚えれば足りる。

| 聞きたいこと | 表現 |
| --- | --- |
| 別サイズ | **Do you have this in a small / medium / large?** |
| 大きめ・小さめ | **Do you have this in a bigger / smaller size?** |
| 色違い | **Does this come in any other colors?** |
| 特定の色 | **Do you have this in black?** |
| 在庫確認 | **Could you check if you have any in the back?**（在庫を見てきてもらえますか） |

```text
A: Hi there! Can I help you find anything?
B: Yes, actually. Do you have this shirt in a medium?
A: Let me check in the back. What color were you looking for?
B: Black, if you have it.
A: You're in luck — here you go.
```

（A: いらっしゃいませ! お探し物はありますか / B: ちょうど探していて。このシャツのMはありますか / A: 在庫を見てきますね。色のご希望は? / B: あれば黒で / A: ありましたよ — どうぞ）

## 試着する

- **Can I try this on?** — 試着してもいいですか
- **Where's the fitting room?** — 試着室はどこですか（**dressing room** とも言う）
- **It's a bit tight. Do you have one size up?** — 少しきついです。ワンサイズ上はありますか
- **It fits perfectly.** — ぴったりです
- **I'll take it.** — これにします／買います

> ⭐ 買うと決めたときの一言は **I'll take it.**。逆に見送るときは **I'll think about it, thanks.**（ちょっと考えます）で角が立たない。

## 値段・会計・返品

- **How much is this?** — これはいくらですか
- **Is this on sale?** — これはセール品ですか
- **Do you take credit cards?** — クレジットカードは使えますか
- **Can I get a gift receipt?** — ギフト用レシートをもらえますか
- **I'd like to return this.** — これを返品したいのですが
- **Can I get a refund?** — 返金してもらえますか
- **Could I exchange this for a different size?** — 別のサイズに交換できますか

```text
A: Hi, I'd like to return this. It didn't fit.
B: No problem. Do you have the receipt?
A: Yes, here it is.
B: Would you like a refund or an exchange?
A: A refund, please.
```

（A: すみません、これを返品したいんです。サイズが合わなくて / B: 承知しました。レシートはお持ちですか / A: はい、これです / B: 返金と交換、どちらになさいますか / A: 返金でお願いします）

> ⚠️ 「返品したい」を直訳して **I want to return this.** と切り出すとぶっきらぼうに響く。**I'd like to return this.** か **Could I return this?** が安全。レシート（**receipt**）が必要なのは日本と同じ。

## まとめ

- 店員の声かけには **I'm just looking, thanks.** — 無視せず一言返すのがマナー
- 探し物は **Do you have... / Do you carry... / Where can I find...?** の3つの型で
- サイズ・色は **in a medium / in black / in any other colors** と in で聞く
- 試着は **Can I try this on?**、買うなら **I'll take it.**、見送るなら **I'll think about it.**
- 返品・返金は **I'd like to return this. / Can I get a refund?** — want ではなく would like で
